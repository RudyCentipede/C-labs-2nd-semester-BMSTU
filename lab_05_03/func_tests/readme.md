# Задание lab_05_03
***
## Входные данные

Двоичный файл, содержащий целые числа, ключ (c, p, s)

## Выходные данные

- **app.exe c number file** - файл, заполненный случайными числами
- **app.exe p file** - числа из файла
- **app.exe s file** - упорядоченный файл


## Тесты
### Негативные:
- **neg_01** - пустой файл (ключ s)
- **neg_02** - некорректное количество ключей
- **neg_03** - пустой файл (ключ p)
- **neg_04** - нет чисел в файле


### Позитивные:
- **pos_01** - целые числа (ключ p) 
- **pos_02** - целые числа (ключ s)
- **pos_03** - одно отрицательное число (ключ p)
- **pos_04** - одно отрицательное число (ключ s)
- **pos_05** - разное число пробельных символов (ключ p)
